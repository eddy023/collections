--- layout: default title: Tips for Ubuntun Installation using EasyBCD ---

1 Check hd0:\* using DiskGenius;

2 Format a disk, leave it unused and unseen by windows;

3 Before setting out to install ubuntu, open a terminal and umount isodevice first: "sudo umount -l /isodevice", or the installation will be blocked at "detecting file system"

4 Divide the disk into 3 subs, first for "\" as primary, second for "home" as login and third for swap area.
