FIND_PACKAGE(Eigen3 REQUIRED): find package pre-installed, generate variable in the form ${Eigen3_LIBS}

INCLUDE_DIRECTORIES: directories of header files (the searching range for header files), those header files included in the compiling source files should be under these directories

LINK_DIRECTORIES: directories of libs (the searching range for lib files)

ADD_EXECUTABLE: generate the target obj or exec file

TARGET_LINK_LIBRARIES: link libs found in "package" or "link_directories" to the target obj
